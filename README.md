# My DSA Journey

I solve problems from various platforms, mainly [LeetCode](https://leetcode.com), and document every solution clearly and consistently.

---

## 🎯 Goals
- Solve **2–3 problems daily** to build consistency  
- Write **clean, readable, and minimal code**  
- Maintain a **well-organized, category-wise structure**  
- Review and revisit previous problems to improve

---

Each problem file contains:
- Problem explanation  
- Problem link  
- My solution with comments  
- ⏱Time and space complexity  
- Notes or edge cases (if any)
